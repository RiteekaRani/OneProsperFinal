# Videos Folder

This folder is for optional local video files.

## How to Add Videos

### Option 1: YouTube Embed (Recommended)
- Use the YouTubeVideo component in your code
- Paste the YouTube video URL or ID
- See component comments for instructions

### Option 2: Local Video Files
1. Upload video files to this folder
2. Reference them using: `/videos/your-video-name.mp4`
3. Use standard HTML5 `<video>` tag

## Video Guidelines
- Recommended formats: MP4, WebM
- Keep file sizes reasonable for web performance
- Consider using YouTube/Vimeo for better performance
